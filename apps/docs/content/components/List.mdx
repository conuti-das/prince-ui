# List

Die List ist eine gruppierte Liste im macOS-Stil: gerahmte Zeilen mit Leading-Icon, Titel, Untertitel und einem Trailing-Slot für Status oder Wert.

## Verwendung

Verwende `List` für eine kompakte, gruppierte Aufzählung mit optionaler Gruppenüberschrift (`label`). Jede Zeile ist eine `ListRow` mit `leading` (Icon/Avatar), `title`, optionalem `subtitle` und `trailing` (Badge, Wert oder Chevron). Pressbare Zeilen bekommen `onPress` und können über `isActive` als ausgewählt markiert werden; mit `inset` rückt die Liste in einen abgerundeten Karten-Container ein.

<DoDont kind="do">Pro Zeile genau eine primäre Aktion über `onPress` — der Trailing-Slot zeigt Status oder Wert, keine zweite Schaltfläche.</DoDont>
<DoDont kind="dont">Statische Zeilen pressbar wirken lassen: ohne `onPress` keinen Hover-/Chevron-Hinweis setzen, sonst erwarten Nutzer eine Navigation.</DoDont>

## Beispiele

Pressbare Zeilen mit Auswahl, Status-Badges und deaktivierter Zeile:

<Example code={`<List label="Marktpartner">
  <ListRow
    leading={<Icon name="building" />}
    title="Stadtwerke München"
    subtitle="9900123000007 · LF"
    trailing={<Badge tone="green">OK</Badge>}
    onPress={() => {}}
    isActive
  />
  <ListRow
    leading={<Icon name="building" />}
    title="Netze BW"
    subtitle="9900456000003 · VNB"
    trailing={<Badge tone="orange">Wartet</Badge>}
    onPress={() => {}}
  />
  <ListRow
    leading={<Icon name="building" />}
    title="E.ON Energie"
    subtitle="Deaktiviert"
    trailing="›"
    onPress={() => {}}
    isDisabled
  />
</List>`} />

Statische, eingerückte Liste als Detail-Kachel:

<Example code={`<List label="Statisch (nicht pressbar)" inset>
  <ListRow title="Sparte" trailing="Strom" />
  <ListRow title="Prozess" trailing="UTILMD" />
  <ListRow title="Betrag" trailing={<Amount value={-129.9} currency="EUR" colored signed />} />
</List>`} />

## Accessibility

Pressbare `ListRow`-Zeilen nutzen React Aria und sind damit per Tastatur fokussierbar und mit Enter/Space auslösbar; der Fokus-Ring kommt aus den prince-ui-Tokens. `isDisabled` nimmt die Zeile aus dem Tab-Fluss und kennzeichnet sie als deaktiviert. Achte darauf, dass ein rein dekoratives Trailing-Chevron (`›`) keine eigene Bedeutung trägt — die Aktion liegt auf der ganzen Zeile, nicht auf dem Pfeil. Statische Zeilen ohne `onPress` bleiben reiner Text und werden nicht als interaktiv angekündigt.
