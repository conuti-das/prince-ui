## Installation

prince-ui wird als Workspace-Paket konsumiert. Lade einmalig das Tokens- und das Komponenten-CSS am App-Einstieg — danach stehen alle Tokens und Styles bereit:

```ts
import '@conuti-das/prince-ui-tokens/tokens.css';
import '@conuti-das/prince-ui/styles.css';

import { Button } from '@conuti-das/prince-ui';
```

Wickle deine App in den `I18nProvider` von React Aria mit `locale="de-DE"`, damit Datums-, Zeit- und Zahlenkomponenten korrekt lokalisiert rendern. Den Rest erledigen die Tokens.
