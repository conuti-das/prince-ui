# prince-ui

Das gemeinsame React-Design-System der CONUTI-Energiewirtschaftsanwendungen — gebaut auf
[React Aria](https://react-spectrum.adobe.com/react-aria/) für Zugänglichkeit von Grund auf,
gekleidet in die Prince-Designsprache. Eine Token-Basis, drei Themes (Apple Dark, Apple Light,
CONUTI-Community), eine konsistente Komponenten-Bibliothek von Formularen über Tabellen bis zu
Diagrammen und ganzen App-Shells.

## Was prince-ui ausmacht

- **Token-getrieben.** Jede Farbe, Schrift, Linie und Bewegung ist eine `--prn-*`-Custom-Property.
  Komponenten verwenden niemals rohe Hex-Werte — deshalb sind sie automatisch theme-fähig.
- **Zugänglich per Default.** Auf React Aria aufgebaut: Tastaturbedienung, Fokus-Management,
  ARIA-Rollen und sichtbare Fokusringe sind eingebaut, nicht nachgerüstet.
- **Drei Themes, ein Code.** Apple Dark, Apple Light und das CONUTI-Community-Styling teilen sich
  denselben Komponenten-Code; gewechselt wird über ein einziges Attribut am `<html>`.
- **Deutsch & lokalisiert.** Datums-, Zahlen- und Beträge-Komponenten laufen mit `de-DE` als Locale.

## Foundations

Die Grundlagen, auf denen jede Komponente steht:

- [Design-Tokens](/foundations/tokens) — die Single Source of Truth (`--prn-*`): Radien, Maße, Bewegung.
- [Farben](/foundations/colors) — semantische Farbrollen: Akzent, Flächen, Text-Hierarchie, Status, Charts.
- [Typografie](/foundations/typography) — die Apple-Typo-Skala als fertige `font`-Tokens.
- [Theming](/foundations/theming) — `setTheme` und die drei Themes im Detail.
- [Icons](/foundations/icons) — der monochrome `<Icon>`-Satz mit `currentColor`.

## Installation

prince-ui wird als Workspace-Paket konsumiert. Lade einmalig das Tokens- und das
Komponenten-CSS am App-Einstieg — danach stehen alle Tokens und Styles bereit:

```ts
import '@conuti-das/prince-ui-tokens/tokens.css';
import '@conuti-das/prince-ui/styles.css';

import { Button } from '@conuti-das/prince-ui';
```

Wickle deine App in den `I18nProvider` von React Aria mit `locale="de-DE"`, damit Datums-,
Zeit- und Zahlenkomponenten korrekt lokalisiert rendern. Den Rest erledigen die Tokens.
