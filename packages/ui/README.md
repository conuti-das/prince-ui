# prince-ui

Prince-Look React-Komponenten auf Basis von [React Aria](https://react-spectrum.adobe.com/react-aria/).
A11y (Fokus-Management, Tastatur-Nav, ARIA) ist eingebaut, nicht nachträglich.

## Nutzung

```ts
import "prince-ui-tokens/tokens.css";
import "prince-ui/styles.css";
import { setTheme } from "prince-ui";
```

## Status

**Phase 0** — nur Gerüst + Token-Re-Export. Ab Phase 1 kommen die L1-Primitives
(Button, Dialog, Menu, Select, Tabs/SegmentedControl, SearchField, Checkbox,
Switch, Tooltip). Roadmap siehe `docs/prince-ui-framework-plan.md` in finops.
